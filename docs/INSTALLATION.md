# Installation

## Download

```sh
npm i -g yarn
git clone git@github.com:ethanneff/react-native-web-typescript.git
cd react-native-typescript
yarn install
```

## Run

```sh
yarn ios
```

```sh
yarn android
```

```sh
yarn web
```

## Lint

```sh
yarn lint
```

## Test

```sh
yarn test
```

## Deploy

```sh
yarn build
```
